auto-py-to-apk

Python kodunu hiç değiştirmeden, bağımlılıklarıyla birlikte çalışan bir Android APK'sına dönüştür.

🎯 v0.1.0 Alpha 🐧 Docker'sız ⚡ QEMU Motor

📦 Terminalden Kurulum

pip install py2apk
py2apk

veya auto-py-to-apk yazarak GUI'yi başlat

✅ Sıfır Değişiklik

Tkinter, Pygame, Flask, NumPy... Koduna dokunma. Olduğu gibi APK olsun.

✅ Docker Yok

QEMU tabanlı taşınabilir build motoru. Docker, WSL, sanal makine kurmana gerek yok.

✅ 60 FPS Oyun

Pygame uygulamaların otomatik pygame_sdl2'ye dönüşür, donanım hızlandırmalı çalışır.

✅ Küçük APK

10-40 MB arası final APK boyutu. Gereksiz şişirme yok, sadece ihtiyacın olan içeride.

✅ Web Sunucusu

Flask, Django, FastAPI uygulamaların WebView içinde Foreground Service ile çalışır.

✅ Komut Satırı

CLI betiklerin TerminalView ile tam stdin/stdout desteğiyle çalışır.

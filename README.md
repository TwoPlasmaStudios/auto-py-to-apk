# auto-py-to-apk (py2apk) 🐍➡️📱

Python kodunu **hiç değiştirmeden**, bağımlılıklarıyla birlikte çalışan bir **Android APK**'sına dönüştür.

## 🎯 Vizyon
`auto-py-to-exe` ne kadar basitse, bu da o kadar basit olacak.  
**Docker yok, WSL yok, sanal makine yok.**

## 🚀 Hızlı Başlangıç

```bash
pip install py2apk
py2apk
